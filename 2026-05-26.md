# 2026-05-26 專案日誌

## 今日進度：雲端部署、LINE 整合與行動端優化

### 1. 雲端部署與 LINE LIFF 整合
- **GitHub Pages 部署**：成功將專案上傳至 GitHub 並透過 GitHub Actions 實現自動打包與發布。
    - 網址：`https://jt-cloud-ops.github.io/jt-cai-quote/`
- **LINE LIFF 整合**：建立 LINE Developers 頻道，將系統正式轉化為 LINE 內置應用程式。
    - LINE 專屬連結：`https://liff.line.me/2010201815-z3mfiA3O`

### 2. 功能增強與行動端專屬功能
- **自訂存檔名稱**：點擊儲存時新增彈出視窗，允許使用者在下載 `.json` 前自訂檔名。
- **LINE 分享功能**：在報價單底部新增「分享至 LINE」綠色按鈕，點擊可直接將報價摘要傳送至 LINE 好友或群組（已整合 LIFF SDK）。
- **版本識別機制**：在系統載入畫面加入版本編號（如：0526-1750），方便確認手機/雲端版本是否已同步更新。
- **行動端版面修復**：
    - **主選單優化**：修復 4 家公司按鈕在手機上跑版的問題，改為自動換行的 2x2 格狀佈局。
    - **操作按鈕優化**：將底部的「儲存、列印、分享」按鈕在手機上改為垂直排列，確保不被裁切且容易點擊。

### 3. 金額計算與稅制優化
- **小計手動輸入**：單張類報價支援手動輸入小計金額，並具備「自動計算/手動輸入」優先級邏輯。
- **稅制自動反推**：當選擇「含稅」時，系統會自動從小計金額反推未稅合計與 5% 稅金，確保總額一致。
- **排版與細節微調**：
    - 找回遺失欄位：修復報價單底部「印訂日期」與「交貨日期」顯示。
    - 金額與稅制合併：將 (含稅/未稅) 標籤移入表格內的「金額」欄位。
    - 單價空白邏輯：單價未輸入時保持空白顯示。

**目前狀態**：系統已成功上線至雲端並進入 LINE 行動化階段，解決了大部分的手機相容性與排版問題。
