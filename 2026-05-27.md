# 2026-05-27 專案日誌

## 今日進度：LINE 功能深度修復、排版優化與存檔增強

### 1. LINE LIFF 功能整合與修復
- **LINE 分享功能啟動**：
    - 正式導入 LINE LIFF SDK。
    - 實作 `shareTargetPicker` 功能，點擊綠色分享鈕可彈出好友/群組選擇器，發送報價摘要。
    - 加入登入檢查機制，確保在 LINE 環境下能正確抓取權限。
- **列印限制解決方案**：
    - 針對 LINE 內部瀏覽器不支援 `window.print()` 的限制，加入自動提示導引，引導使用者至外部瀏覽器（Safari/Chrome）進行 PDF 列印。

### 2. 資料儲存與下載優化
- **自訂檔名彈窗**：
    - 點擊「儲存此報價單」時，新增彈出視窗詢問存檔名稱。
    - 預設帶入 `報價資料_客戶_印件` 格式，方便使用者分類管理下載的 `.json` 備份。
- **小計金額手動輸入優先**：
    - 完善「手動輸入小計」邏輯，系統會優先檢查手動輸入值，若無則恢復自動計算（數量x單價），確保報價靈活性。

### 3. 行動端版面與 UI 最終修復
- **主選單自適應**：
    - 強制修復手機版 4 家公司按鈕排版，確保採 2x2 格狀排列且不溢出底色方框。
- **遺失欄位找回**：
    - 重新校正 `QuotationPreview` 邏輯，確保表格下方的「印訂日期」與「交貨日期」在任何模式下皆能正確顯示。
- **極致單頁壓縮**：
    - 優化 CSS Print 樣式，將邊距縮減至 0.5cm 並調整行高，確保多品項內容仍能塞入一頁 A4。
