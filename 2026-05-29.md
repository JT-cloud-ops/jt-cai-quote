# 2026-05-29 專案日誌

## 今日進度：報價內容數位化、LINE 分享專業化與安全性分析

### 1. LINE 分享功能大升級 (數位報價單卡片)
- **詳細摘要訊息**：點擊「分享至 LINE」後，系統不再只傳送網址，而是會自動整理出一份包含「報價公司、客戶名稱、所有印件品項規格、總計金額（含/未稅）」的精美文字摘要。
- **數位孿生機制**：分享訊息的內容與 PDF 報價單完全一致，讓接收者不需點開附件即可掌握核心報價。
- **一鍵匯入功能優化**：
    - 強化 URL 參數解碼邏輯，解決 LINE 連結折行導致的資料損壞問題。
    - 加入「匯入成功通知」，當使用者點擊連結進入系統後，會彈窗提示匯入完成。

### 2. 計算與邏輯優化
- **金額計算精準化**：重構 `QuotationPreview` 計算引擎，確保單項「含稅/未稅」選擇與底部總計金額絕對對應。
- **自動換行優化**：針對表格內「(含稅)」標籤設定 inline-block，確保在任何螢幕寬度下，稅制標籤都能完整換行，不被拆散。

### 3. 安全性評估
- **架構安全性確認**：分析並確認系統資料採「本地儲存」模式，GitHub 雲端不儲存任何客戶隱私。
- **風險分析**：確認主要風險僅限於「分享連結」的傳遞範圍，並提供未來加密或私有化部署的建議。

### 4. 版本管理與雲端同步
- **GitHub 全量同步**：所有修正與新功能已成功 push 至 GitHub 倉庫，LINE 版本已同步至最新。
