# 2026-05-22 專案日誌

## 今日進度：估價單系統功能大躍進 (ERP 雛形) 與架構模組化

### 1. 核心功能更新 (quotation-app)
- **多品項支援**：表單可新增/刪除多個印件，自動加總金額、計算稅額。
- **單位選擇器**：數量欄位加入「份、張、本、個」下拉選單。
- **結構化日期**：訂印與交貨日期改為「年/月/日」分離輸入，並統一顯示於底部條款區（移除頂部重複顯示）。
- **表格高度固定**：表格維持至少 7 行，不足自動補空白，格式更整齊。
- **特別色支援**：全系統新增「特別色」輸入欄位，報價單會以 `印色 (特別色)` 的格式括號並列顯示。
- **其他備註位置調整**：將整份報價單的備註獨立顯示於合計金額上方。

### 2. 存取與使用者體驗優化 (UX)
- **UI/UX 強化**：
    - 美化了列印按鈕，並針對 A4 比例進行了 CSS 排版優化，確保列印 PDF 時美觀整齊，且強制縮放於一頁內。
    - **快捷輸入優化**：全表單支援「按下 Enter 鍵自動跳至下一個輸入框」，大幅提升資料建檔與報價速度（多行文字框保留換行功能）。
- **主選單頁面**：新增系統入口網頁，提供直覺的 UI 進行三大報價單類型選擇。
- **三大類型支援**：
    - **單張類**：標準單一品項報價。
    - **冊子類**：支援封面、扉頁、內頁等結構化零件管理，首列顯示名稱/開數/裝訂並垂直合併總價，後續列齊右顯示明細。
    - **百貨類**：繼承冊子架構，並強制新增「總公司量」發貨欄位，自動顯示於明細最末列。
- **全局業務設定**：在主選單即可設定業務人員資訊，自動同步至所有報價類型中，具備 LocalStorage 自動記憶功能。

### 3. 資料庫與檔案管理
- **客戶資料庫系統**：自動收集儲存客戶聯絡資訊，輸入名稱時自動彈出選單一鍵帶入歷史資料。
- **歷史紀錄系統**：瀏覽器內建 20 筆歷史紀錄管理。
- **雙重存檔功能**：「儲存此報價單」會同步存入歷史紀錄並下載 `.json` 備份檔；「匯入檔案」可讀取備份檔進行修改再次報價。

### 4. 列印與程式架構優化
- **極致 A4 優化與自適應排版**：
    - 調整字體與間距，確保多品項時仍能縮放於一頁 A4 內。
    - **重構 CSS Grid 排版**：強制保留右側預覽區 `21cm` (A4真實寬度)，左側表單自適應伸縮；解決畫面裁切與捲軸跳動問題，並支援小螢幕自動上下排列。
- **自動建議檔名**：根據報價單內容自動生成下載檔名。
- **跨機相容修正**：移除 `crypto.randomUUID` 依賴，確保 HTTP IP 連線環境下運作正常。
- **程式碼模組化 (Refactoring)**：將龐大的表單與預覽元件，依據三大報價類型拆分為獨立的子模組，大幅提升程式碼的可讀性與未來維護性。

**目前狀態**：系統已完成「單張類」、「冊子類」、「百貨類」三大商業報價模組開發與架構重構，具備 ERP 等級的客戶與業務資料管理，支援檔案雙重備份，排版精美且完全符合 A4 列印規範，專案完美收尾。
