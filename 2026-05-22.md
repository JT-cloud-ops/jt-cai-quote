# 2026-05-22 專案日誌

## 今日進度：估價單系統功能大躍進 (ERP 雛形)

### 1. 核心功能更新 (quotation-app)
- **多品項支援**：表單可新增/刪除多個印件，自動加總金額、計算稅額。
- **單位選擇器**：數量欄位加入「份、張、本、個」下拉選單。
- **結構化日期**：訂印與交貨日期改為「年/月/日」分離輸入與顯示。
- **表格高度固定**：表格維持至少 7 行，不足自動補空白，格式更整齊。
- **其他備註位置調整**：將整份報價單的備註獨立顯示於合計金額上方。

### 2. 資料管理與資料庫
- **客戶資料庫系統**：
    - 自動收集儲存客戶聯絡資訊。
    - 輸入名稱時自動彈出選單，一鍵帶入歷史資料。
- **歷史紀錄系統**：
    - 瀏覽器內建 20 筆歷史紀錄管理。
    - 業務人員資訊（姓名、電話）跨頁面自動記憶。
- **雙重存檔功能**：
    - 「儲存此報價單」同步存入歷史紀錄並下載 `.json` 備份檔。
    - 「匯入檔案」功能：可讀取備份檔進行修改再次報價。

### 3. 列印與跨機優化
- **極致 A4 優化**：調整字體與間距，確保多品項時仍能縮放於一頁 A4 內。
- **自動建議檔名**：根據報價單內容自動生成下載檔名。
- **跨機相容修正**：移除 `crypto.randomUUID` 依賴，確保 HTTP IP 連線環境下運作正常。

### 4. 版本管理
- **Git Commit**：已將所有功能整合並提交至 Git。

### 5. 系統架構與入口網頁 (Dashboard)
- **主選單頁面**：新增系統入口網頁，提供直覺的 UI 進行報價單類型選擇。
- **類型支援**：預留「單張類」、「冊子類」、「百貨類」三大報價路徑。
- **全局業務設定**：在主選單即可設定業務人員資訊，並自動同步至所有報價類型中，具備 LocalStorage 自動記憶功能。
- **導覽系統**：實現頁面切換邏輯，各報價頁面均可一鍵返回主選單。

---
**目前狀態**：系統已具備完整的 ERP 入口架構，並完成「單張類報價」的全功能開發，後續將針對不同類別進行排版擴充。
